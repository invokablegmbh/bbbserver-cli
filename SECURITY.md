# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly and do **not** disclose it publicly before it has been reviewed and fixed.

Please use the official IT security contact/process of Invokable GmbH:

- https://invokable.gmbh/en/it-security

## What to Include

To help us triage quickly, include as much of the following as possible:

- A clear description of the issue and potential impact
- Steps to reproduce (proof of concept, if possible)
- Affected version(s), environment, and platform details
- Suggested remediation (if available)

## Response Expectations

We will acknowledge receipt and aim to investigate promptly. Depending on complexity, remediation timelines may vary.

## Scope

This policy applies to this repository and its published binaries.
